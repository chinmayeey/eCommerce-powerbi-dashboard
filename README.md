# 🛒 Indian eCommerce Orders Data Analysis - Power BI Dashboard

## 📊 Overview
This project presents an interactive Power BI dashboard analyzing Indian eCommerce sales data.

## 🔍 Key Insights
- Total Sales: 12.01M
- Total Quantity: 40K
- Total Profit: 286.40K
- Average Order Value: 229.86

## 📈 Features
- Sales by Category
- Profit by Sub-Category
- Sales by Region & State
- Segment Analysis (Consumer, Corporate, Home Office)
- Dynamic Filters (Category, State, Region, Ship Mode)

## 🛠 Tools Used
- Power BI
- Excel / CSV
- Data Cleaning & Transformation
